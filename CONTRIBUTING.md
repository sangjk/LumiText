# Contributing to LumiText

Thanks for your interest in contributing!

## Reporting Issues

Found a bug or have a feature idea? Please open an issue using one of the templates:

- [🐛 Bug Report](https://github.com/lumitext/lumitext/issues/new?template=bug_report.md)
- [✨ Feature Request](https://github.com/lumitext/lumitext/issues/new?template=feature_request.md)

## Guidelines

- Search existing issues before opening a new one.
- Be clear and descriptive — include steps to reproduce, screenshots, and your macOS environment.
- For any security-related issues, please refer to [SECURITY.md](SECURITY.md).

## Code Contributions

LumiText is a closed-source macOS application distributed via the Mac App Store. This repository serves as the project's public home for documentation, issue tracking, and community discussion. We welcome bug reports, feature suggestions, and documentation improvements.

## Community

- Be respectful and constructive.
- Help others by answering questions in issues.
- Share how you use LumiText — we love hearing real-world use cases.

Thanks for helping make LumiText better!
