## 🎯 LumiText について

LumiText は、軽量かつパワフルな **macOS ツール**です。画面上のあらゆる場所（画像、PDF、動画、Web サイト、一時停止した動画のフレームなど）からテキストをキャプチャし、即座に認識・翻訳・コピーします。手動で文字を打ち直す必要はありません。すべての処理は Mac 上でローカルに実行されます。

### ✨ 主な機能

- **📸 スクリーンショット文字認識** — ショートカットを押して画面上の任意の領域を選択すると、テキストが自動認識されクリップボードにコピーされます。全画面および範囲指定に対応。
- **🌍 多言語 OCR** — 中国語（簡体字/繁体字）、英語、日本語、韓国語、フランス語、ドイツ語、スペイン語、ポルトガル語、ロシア語、アラビア語などに対応。言語の自動検出で手動切り替え不要。
- **🔄 インスタント翻訳** — 認識テキストをワンクリックで複数言語に翻訳。原文と翻訳をシームレスに切り替え。
- **📱 QR コード・バーコード** — QR、Aztec、DataMatrix、PDF417、Code128、Code39、EAN13、EAN8 などを画面上から直接スキャン。

## 🏆 LumiText の利点

**⚡ 高速パフォーマンス**
macOS ネイティブ開発により、Apple Silicon（M1/M2/M3/M4）の性能を最大限活用。ミリ秒単位の文字認識を実現。クラウド遅延なし、すべてローカル処理。

**🔒 プライバシー重視**
すべての OCR 処理は Mac 上でローカル実行。スクリーンショットや認識テキストがデバイス外に出ることはありません。コア機能は完全オフラインで動作。

**🎯 シンプルで邪魔にならない**
メニューバーに静かに常駐し、グローバルホットキーでいつでも呼び出せます。認識結果は自動コピー。複雑な設定不要、すぐに使えます。

**🌐 30 の UI 言語**
中国語、英語、日本語、韓国語、フランス語、ドイツ語、スペイン語、ポルトガル語、アラビア語など 30 言語に対応。システム言語に自動適応。

## 🚀 使い方

**① ⌘ ⇧ S を押す** — ブラウザ、PDF リーダー、動画プレイヤーなど、どのアプリからでも呼び出し可能。

**② 領域を選択** — マウスでテキスト領域をドラッグ。全画面または範囲指定に対応。

**③ 貼り付け** — 認識テキストが自動コピーされます。⌘V で任意のアプリに貼り付け。

## 🎯 クイックスタートガイド

1. **📲 ダウンロード＆インストール** — Mac App Store から LumiText を入手
2. **🔐 権限の付与** — 初回起動時に「画面収録」と「アクセシビリティ」権限をシステム環境設定で許可
3. **⌨️ 使い始める** — `⌘⇧S` を押すだけで、どのアプリからでも LumiText を起動

### キーボードショートカット

| ショートカット | 機能 |
|---------------|------|
| `⌘⇧S` | スクリーンショット OCR 認識 |
| `⌘⇧V` | クリップボード履歴 |
| `⌘⇧T` | 翻訳切り替え |
